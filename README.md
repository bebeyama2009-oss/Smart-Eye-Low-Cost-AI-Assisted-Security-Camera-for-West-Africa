# Smart Eye 👁️

*Motion-triggered security camera with Telegram alerts — built on ESP32-CAM**

> $25 hardware · Wi-Fi connected · Sends photos to your phone instantly

## What it does
When the PIR sensor detects motion, Smart Eye wakes up, takes a photo with the OV2640 camera, saves it to the microSD card, and sends it to your Telegram chat within seconds.

## Why it exists
Professional security cameras cost $200–500 in West Africa. This proves you can build a working alternative for $25.

## Hardware
- ESP32-CAM (AI-Thinker) + OV2640 camera
- HC-SR501 PIR motion sensor
- Active buzzer + LED indicator
- MicroSD card (8GB)
- USB power supply
